onVScroll がスクロールを監視し、scrollHeight − scrollTop − clientHeight で下端到達を判定して loadNext を呼ぶ。loadNext は loading フラグで多重実行を防いだうえで、allTasks.slice で次の50件を取り出し、tasks.push と groupsDS.add / itemsDS.add で表示に追加する。続いて tasks.length − MAX_LOADED が正なら、tasks.splice(0, over) で先頭を切り、その id を groupsDS.remove / itemsDS.remove で破棄し、windowStart を前進させる。最後に requestAnimationFrame 内で el.scrollTop を破棄分だけ戻してジャンプを防ぐ。上端到達時は loadPrev が同じ流れを反転（tasks.unshift で前方追加、末尾を splice、scrollTop を加算）して実行する。
